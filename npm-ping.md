npm-ping(1) -- Ping npm 注册表
================================
基于 [npm-ping(1)](https://github.com/npm/npm/blob/latest/doc/cli/npm-ping.md) 官方文档翻译版本

## SYNOPSIS 概要
```bash
npm ping [--registry <registry>]
```


## DESCRIPTION 描述
Ping 已配置的或给定的 npm 注册表并进行身份验证。
