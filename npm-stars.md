npm-stars(1) -- 查看 star 过的包
=================================================
基于 [npm-stars(1)](https://github.com/npm/npm/blob/latest/doc/cli/npm-stars.md) 官方文档翻译版本

## SYNOPSIS 概要
```bash
npm stars [<user>]
```


## DESCRIPTION 描述
如果你 strar 了很多不错的包，这个命令可以让你快速找到它们。

这个命令也可以查看某个用户 star 过的包。
