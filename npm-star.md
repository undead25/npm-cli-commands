npm-star(1) -- 标记你喜欢的包
==========================================
基于 [npm-star(1)](https://github.com/npm/npm/blob/latest/doc/cli/npm-star.md) 官方文档翻译版本

## SYNOPSIS 概要
```bash
npm star [<pkg>...]
npm unstar [<pkg>...]
```
## DESCRIPTION 表示

“Star” 一个包意味着你对它有兴趣。这是一种表达你关心的方式。

“Unstar” 则与上面相反。

这是一个布尔的事情，要么 star 要么 unstar。重复 star 是无效的。
