npm-whoami(1) -- 显示 npm 用户名
=====================================
基于 [npm-whoami(1)](https://github.com/npm/npm/blob/latest/doc/cli/npm-whoami.md) 官方文档翻译版本

## SYNOPSIS 概要
```
npm whoami [--registry <registry>]
```


## DESCRIPTION 描述
打印 `username` 配置。
