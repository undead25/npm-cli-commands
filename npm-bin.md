npm-bin(1) -- 显示 npm bin 文件夹
====================================
基于 [npm-bin(1)](https://github.com/npm/npm/blob/latest/doc/cli/npm-bin.md) 官方文档翻译版本

## SYNOPSIS 概要
```bash
npm bin [-g|--global]
```

## DESCRIPTION 描述 

打印 npm 将安装可执行文件的文件夹。

译者注：
```bash
npm bin -g
/usr/local/bin  # mac print
```
