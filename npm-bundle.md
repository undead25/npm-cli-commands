npm-bundle(1) -- 已删除
========================
基于 [npm-bundle(1)](https://github.com/npm/npm/blob/latest/doc/cli/npm-bundle.md) 官方文档翻译版本

## DESCRIPTION 描述

`npm bundle` 命令在 1.0 中已被删除，原因很简单，已经不需要它了，因为现在的默认行为是安装到本地空间中。

现在只需使用 `npm install` 来做 `npm bundle` 以前做的工作。
