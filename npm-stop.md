npm-stop(1) -- stop 脚本命令
=============================
基于 [npm-stop(1)](https://github.com/npm/npm/blob/latest/doc/cli/npm-stop.md) 官方文档翻译版本

## SYNOPSIS 概要
```bash
npm stop [-- <args>]
```


## DESCRIPTION 描述
执行包 npm 脚本中的 `stop` 命令。
