npm-test(1) -- test 脚本命令
=============================
基于 [npm-test(1)](https://github.com/npm/npm/blob/latest/doc/cli/npm-test.md) 官方文档翻译版本

## SYNOPSIS 概要
```
npm test [-- <args>]

aliases: t, tst
```


## DESCRIPTION 描述
执行包 npm 脚本中的 `test` 命令。
