npm-root(1) -- 显示 npm 根目录
===============================
基于 [npm-root(1)](https://github.com/npm/npm/blob/latest/doc/cli/npm-root.md) 官方文档翻译版本

## SYNOPSIS 概要
``` bash
npm root [-g]
```

## DESCRIPTION 描述
将有效的 `node_modules` 文件夹打印出来。
